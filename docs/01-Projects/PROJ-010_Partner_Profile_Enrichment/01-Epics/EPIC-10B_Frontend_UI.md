# EPIC-10B: Interface do Aluno (Frontend)

**Projeto Pai:** PROJ-010
**Status:** Planejado

## Descrição
Desenvolvimento da nova página de Perfil do Parceiro no aplicativo do aluno (`/student/[slug]/partner/[id]`). Foca na experiência de usuário, visualização das informações ricas e conversão (cliques em contatos).

## Histórias Contidas

### STORY-10B-01: Layout da Página de Perfil
*   **Objetivo:** Criação da estrutura base da página, Header com Galeria e Seções de Informação.

### STORY-10B-02: Lógica de Negócio e Interatividade
*   **Objetivo:** Implementar a lógica de "Aberto/Fechado", botões de ação (Deeplinks) e exibição dinâmica de ícones.

## Dependências
*   EPIC-10A concluído (Dados disponíveis no Banco).
